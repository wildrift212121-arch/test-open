# Denis Bot — Aion 2 Route Recorder

Пассивный рекордер маршрутов для Aion 2 с последующим воспроизведением через DD драйвер.

## Возможности (v4.0)
- Запись действий игрока в лог‑файл
- Подсчёт событий
- Отображение длительности записи
- Управление через удобный GUI
- Поддержка DD кодов клавиш
- Логи в формате UTF‑8

## Ограничения
DD драйвер НЕ умеет считывать состояние клавиш.  
Он может только эмулировать нажатия.  
Поэтому рекордер работает в пассивном режиме.

## Как использовать
1. Поместите dd60300.dll в папку со скриптом  
2. Запустите скрипт от имени администратора  
3. Запустите Aion 2  
4. Нажмите «Старт записи»  
5. Переключитесь в игру и играйте вручную  
6. Нажмите «Стоп записи»  
7. Лог появится в папке со скриптом  

## Формат лог‑файла
[START_TIMESTAMP:1704067200]
0.000:602:DOWN
1.500:602:UP
2.100:801:DOWN
